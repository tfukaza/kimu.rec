The Apple Magic Keyboard with Touch ID is the keyboard pick, and yes, this is a controversial one. Keyboard people will tell you to fix comfort with a wrist rest, a taller case, a different switch, or a more elaborate desk setup. The better question is: what if the keyboard was already low enough that you did not need to contort your wrists around it?

That is the point of the Magic Keyboard. It is thin, stable, precise, and low-travel. The shallow profile keeps your hands closer to the desk surface, which can make it easier to type without adding a wrist rest just to make the keyboard usable.

## Things to consider

- This is not a mechanical keyboard enthusiast pick. It does not feel like an MX Red, and it is not a linear switch board.
- It might not be the best keyboard for every gaming setup, but the low travel is at least directionally right for fast key presses.
- Touch ID is genuinely useful if you use an Apple silicon Mac.
- The USB-C version pairs and charges with the same modern cable ecosystem as current Macs.
- The big caveat is platform fit: this is best if you are already in the Mac ecosystem.
